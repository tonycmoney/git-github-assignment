# git-github-assignment

A simple HTML personal homepage demonstrating Git and GitHub workflow.

## Description
A single-page website with a heading and a graduation cap image. Built as part of a Git and GitHub assignment for software development coursework.

## Files
- index.html: main HTML file with embedded CSS styling
- cap.jpg: graduation cap image displayed on the page
- README.md: this file

## Setup
1. Clone the repository:
   git clone https://github.com/tonycmoney/git-github-assignment.git
2. Open index.html in a browser, or use the Live Server extension in VS Code.

## Branch Workflow
This project demonstrates a feature branch workflow:
1. Initial files committed to main
2. Subtitle added on feature-branch
3. Pull request opened and merged into main
